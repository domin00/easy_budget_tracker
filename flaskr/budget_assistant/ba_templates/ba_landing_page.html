Welcome to the Budget Assistant App

The Budget Assistant App helps you effortlessly manage and categorize your expenses and transactions. If you find it challenging to consolidate spending information from multiple banking providers, this app is the solution you might need.

What does it do?

The Budget Assistant App allows you to upload bank statements, categorize transactions, and gain a comprehensive overview of your spending. It simplifies the process of managing expenses across various banking platforms, providing a centralized and user-friendly interface.

How was it built?

The Budget Assistant App is built using Flask, a powerful web framework in Python. We leverage the flexibility and efficiency of Flask to handle server-side logic and routing. The app's data is stored in a SQL database, ensuring robustness and scalability.

Our technology stack includes Python, Flask, SQLAlchemy for database management, and other libraries to enhance functionality. The frontend is designed with HTML, CSS, and potentially JavaScript for dynamic features.

Additional Features

The Budget Assistant App provides features such as customizable expense categories, insightful visualizations, and the ability to analyze spending trends over time. It's a comprehensive tool designed to make financial management seamless.
